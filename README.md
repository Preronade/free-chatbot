# 🤖 Free AI Chatbot

A simple full-stack chatbot powered by Google's Gemini 2.5 Flash API — free to run with no card required (500 requests/day on Gemini's free tier).

## Tech Stack
- **Backend:** FastAPI (Python)
- **Frontend:** Vanilla HTML/CSS/JS
- **AI Model:** Gemini 2.5 Flash via `google-generativeai`
